# Subscription Model

PREDIKT v0.3 adds a plan foundation without billing integration.

Seeded plan families:

- Free
- Creator Starter
- Creator Pro
- Creator Max
- Brand Starter
- Brand Growth
- Enterprise/API

Plan controls currently support:

- room limits
- participant limits
- custom branding eligibility
- sponsored room eligibility
- analytics eligibility
- Drops eligibility

No wallet, billing, or payment processing is included in MVP+.
