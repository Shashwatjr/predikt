# Admin Wireframes

Planned lightweight screens:

- AdminLogin
- AdminDashboard
- Users
- Creators
- Campaigns
- Sponsors
- Drops
- PrivacyRequests
- AuditLogs
- Compliance
- AiGovernance
- Settings

Suggested MVP flow:

1. Login
2. Dashboard summary cards
3. Left navigation for users, creators, sponsors, campaigns, drops, privacy, audit, AI
4. Table-first views with simple filter/search
5. Detail drawer or detail page for update actions
