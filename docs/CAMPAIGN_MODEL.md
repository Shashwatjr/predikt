# Campaign Model

Campaign foundations link sponsors, creators, rooms, and aggregate metrics.

Primary models:

- `Sponsor`
- `Campaign`
- `CampaignMetric`

Current campaign metrics:

- impressions
- joins
- predictionsSubmitted
- uniqueParticipants
- shares
- dropsUnlocked
- dropsRedeemed
- resultCardsShared

This is a backend foundation for future sponsor reporting and creator campaign operations.
