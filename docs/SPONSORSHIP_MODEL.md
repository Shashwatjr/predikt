# Sponsorship Model

PREDIKT v0.3 includes a sponsorship foundation at the room level.

Current room sponsor fields:

- `isSponsored`
- `sponsorName`
- `sponsorLogoUrl`
- `sponsorBrandColor`
- `sponsorTagline`
- `resultCardSponsorText`

Rules:

1. Sponsored rooms must clearly show `Sponsored Room`.
2. Sponsored rooms may show `Powered by {sponsorName}`.
3. Sponsored branding must never weaken safety messaging.
4. Sponsored Drops must show `Terms apply`.
5. Sponsored language must not imply cash value, payout, or winnings.

Demo guidance:

- Use generic placeholders like `Travel Partner`, `Ride Partner`, or `Food Delivery Partner`.
- Use real brand assets only with permission.
